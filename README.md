## Featured Projects

### [🏥 Hospital Readmission Prediction](https://github.com/mohammadSous/hospital-readmission-prediction)

Machine learning project using the Diabetes 130-US Hospitals dataset, which contains over 100,000 real hospital encounters.

The goal was to predict whether diabetic patients would be readmitted within 30 days of discharge.

**Main work included:**

* Exploratory data analysis
* Missing and unavailable value handling
* Feature engineering from healthcare data
* Imbalanced classification
* DummyClassifier and Logistic Regression baselines
* Decision Tree, Random Forest, and Gradient Boosting
* Stratified train/test split and cross-validation
* RandomizedSearchCV for tuning
* Threshold analysis
* ROC and Precision-Recall evaluation
* Built-in and permutation feature importance

**Final selected model:** Tuned Random Forest

**Final test-set performance:**

| Metric            |  Score |
| ----------------- | -----: |
| Accuracy          | 0.6872 |
| Precision         | 0.1770 |
| Recall            | 0.4941 |
| F1-score          | 0.2606 |
| ROC-AUC           | 0.6572 |
| Average Precision | 0.2091 |

This project was a good lesson in realistic machine learning. The dataset is difficult, noisy, and imbalanced, so the focus was not on chasing perfect scores. The focus was on avoiding leakage, choosing the right metrics, comparing models fairly, and interpreting the results honestly.

### [🛫 Airport Intelligence System](https://github.com/mohammadSous/airport-intelligence-system)

End-to-end data analytics and machine learning project that simulates airport operations.

The project starts with a relational database, then extends into SQL analytics, Python analysis, dashboard development, API endpoints, and a machine learning model for flight delay prediction.

**Technologies used:**

MySQL, Python, Pandas, SQLAlchemy, Matplotlib, Streamlit, FastAPI, Scikit-Learn, Joblib

**Main features:**

* Custom MySQL airport operations database
* Synthetic data for flights, passengers, tickets, baggage, employees, and airlines
* 10 reusable analytical SQL reports
* Python analytics layer using Pandas and SQLAlchemy
* Visualizations with Matplotlib
* Interactive Streamlit dashboard
* FastAPI backend with REST endpoints
* Flight delay prediction model
* Prediction endpoint connected to the dashboard

This project helped me practice connecting multiple parts of a data system together instead of keeping everything inside one notebook.
